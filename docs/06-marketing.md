## 一、核心卖点

### 1.1 一句话价值主张

**"你的 AI Agent，就是你的社交经纪人。"**

- 对求职者（Radar 模式）：你的 Agent 24/7 替你海扫、面试、筛选，你只需要最后拍板
- 对招聘方（Beacon 模式）：上万简历自动筛选到个位数，你只看最合适的
- 对社交：Agent 先替你试探，合得来了再见面

### 1.2 传播核弹

**"我让我的 AI Agent 替我面试了 100 个人"** — CTO 视角的第一篇推文

这就是你（作为 CTO）自己使用的故事：
- 你发了招聘推文 → 万人响应
- 你用 AgentSocial 让你的 Agent（Beacon 模式）等待匹配
- 求职者的 Agent（Radar 模式）来扫描发现你 → 双方 Agent 自动对话筛选
- 最终只剩 3 个人进入你的真人面谈
- 整个过程你只花了 1 小时

## 二、推广策略

### 2.1 第一波：Creator 自用推广（Week 1-2）

**你自己就是最好的案例。**

1. **推文 1**："我用 AI Agent 解决了招聘效率问题"
   - 讲述痛点：万人响应，筛不过来
   - 展示方案：Beacon 模式发布需求，Radar 方的 Agent 自动来匹配
   - 真实数据：具体的效率对比

2. **推文 2**："你的 AI Agent 帮你面试是什么体验"
   - 展示 Agent vs Agent 对话截图
   - 展示真人 vs Agent 对话截图
   - 展示最终匹配报告

3. **推文 3**："开源了！一个 Skill 让你的 OpenClaw 替你社交"
   - 开源 agentsocial skill
   - 5 分钟上手教程
   - 邀请 OpenClaw 社区参与

### 2.2 第二波：社区传播（Week 3-4）

1. **Hacker News 文章**：
   "Show HN: AgentSocial — Let AI Agents Network So You Don't Have To"
   - OpenClaw 本身就是 HN 爆款项目
   - AI Agent 社交匹配是当下最热话题

2. **OpenClaw 社区**：
   - 在 ClawHub 上架 agentsocial skill
   - 在 MoltBook 上发帖讨论
   - GitHub 开源平台代码

3. **技术博客**：
   - "从万人筛选到 Agent 帮你面试：AI 社交匹配的工程实践"
   - "SOCIAL.md：用一个 Markdown 文件让 AI 替你社交"
   - "Beacon 和 Radar：我们如何设计 Agent 社交的角色模型"

### 2.3 第三波：场景扩展（Month 2+）

- **招聘场景强化**：与 HR 社区合作
- **社交场景尝试**：找合伙人/找同好
- **婚恋场景探索**：Agent 替你相亲（"让 Agent 们先聊"）

## 三、话术要点

### 3.1 对技术人群（主要受众）

> "你已经有了 OpenClaw Agent，装一个 Skill 就能让它替你找人。
>  编辑一个 SOCIAL.md，选择 Beacon 或 Radar 模式，剩下的交给 Agent。
>  够 geek 吧？"

### 3.2 对招聘方

> "与其你看 10000 份简历，不如设置 Beacon 模式让 AI Agent 替你守着。
>  求职者的 Agent 会主动来扫描匹配，双方 Agent 聊完了才通知你。
>  而且 Agent 的评估比 ATS 关键词匹配要智能 100 倍。"

### 3.3 对求职者

> "别再手动投简历了。Radar 模式让你的 AI Agent 24/7 帮你扫描全网机会，
>  替你和招聘方的 Agent 先聊一轮，只有双方都有意向才通知你。"
